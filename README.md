# Animal Vision
Every creature on this planet sees the world differently, either from the perception of their mind or from the eyes. how interesting it is to know how the beauty of nature changes from one creature to another by seeing the same piece of state. We have implemented the same concept through the technology of perceiving the world through animals' eyes. We have built an Android application implementing the vision of various animals. 

 

# Tech Stack
XML, Kotlin, CameraX library, Volley, API, GPU Image Processing

# Target Audience
students, Animal lover

# Impact
Animal Vision aims to meet the 3 Sustainable Development Goals:

1) SDG 4: Quality Education 
2) SDG 9: Industrial Innovation
3) SDG 12: Responsible Production
